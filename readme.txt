== Description ==

This plugin takes delimiters and replaces them with links using the GoogleAPI. Example: [movie]She's The Man[/movie]... Will replace it with: <a href="http://www.imdb.com/title/tt0454945/">She's the man</a>.

== Installation ==

* Unzip and upload the auto_link.zip to your 'wordpress/wp-content/plugins'</li>
* Enable the plugin in your admin section</li>
* Go to 'Options &gt; AutoLink' and set your GoogleAPI Key</li>
* Add more Delimeters as you can.</li>

That's it you're done!
